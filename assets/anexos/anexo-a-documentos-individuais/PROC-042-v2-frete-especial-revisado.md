# PROC-042-v2 – Procedimento de Cálculo de Frete Especial (Revisado)

**Versão:** 2.0
**Data de emissão:** 05/02/2024
**Responsável:** Diretoria Comercial
**Status:** Este documento não possui indicação formal de que substitui o PROC-042 v1. Ambos coexistem no SharePoint sem hierarquia clara.

## 1. Objetivo

Definir a fórmula e os parâmetros atualizados para cálculo de frete especial aplicável a cargas com peso acima de 400kg. Os multiplicadores foram revisados para refletir os custos operacionais atualizados de cada região e as alterações na malha de parceiros de transporte.

## 2. Fórmula de cálculo

O frete especial é calculado como:

Valor do frete = Valor base × Multiplicador regional × Fator de peso

Onde:
- Valor base = tarifa publicada na tabela mensal de fretes.
- Multiplicador regional = fator aplicado conforme a região de destino (seção 2.1).
- Fator de peso = 1.0 para cargas de 400kg a 800kg; 1.15 para cargas de 801kg a 2.500kg; 1.40 para cargas acima de 2.500kg.

### 2.1. Multiplicadores regionais (atualizados em fevereiro/2024)

| Região | Multiplicador |
|--------|--------------|
| Sul | 1.25 |
| Sudeste | 1.10 |
| Centro-Oeste | 1.35 |
| Nordeste | 1.45 |
| Norte | 1.70 |

## 3. Prazo de entrega para frete especial

O prazo de entrega para frete especial é calculado como o prazo padrão da rota + 3 dias úteis adicionais para manuseio e roteirização de carga pesada (anteriormente era + 2 dias na versão anterior).

## 4. Condições especiais

- Cargas acima de 4.000kg requerem aprovação prévia do gerente de operações regional.
- Cargas IMO (classes 3, 6 e 8) com peso acima de 400kg seguem tabela específica (PROC-043: Frete de Cargas Especiais). Nota: a PROC-043 está em processo de revisão pelo Compliance e pode sofrer alterações.
- Descontos de volume: a partir de 10 fretes especiais/mês para o mesmo cliente, aplicar desconto de 4% sobre o multiplicador regional. Acima de 18 fretes/mês, desconto de 8%. Descontos maiores requerem aprovação da Diretoria Comercial.

## 5. Disposições transitórias

Durante o período de transição, chamados abertos antes de 01/03/2024 que ainda estejam em processamento devem usar os multiplicadores da versão anterior (PROC-042 v1). Chamados novos a partir de 01/03/2024 devem usar os multiplicadores desta versão.
